# GimiRick

![GimiRick Banner](../GimiRick/Banner.png)

We build open source LLMs and AI systems. Nothing fancy, just solid work on performance and research that actually helps people.

Founded by Mohammad Faiz.

## What we do

* Train and release open source language models
* Research ways to make AI faster and more efficient
* Share everything we build with the community

We believe AI should be open, accessible, and built in public. No gatekeeping, no vague promises, just code and models you can actually use.
